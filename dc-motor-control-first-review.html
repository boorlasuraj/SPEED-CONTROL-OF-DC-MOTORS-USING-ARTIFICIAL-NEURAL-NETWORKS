First Review: Manual Implementation Phase

Current MATLAB Implementation

Circuit Components:

  • Power Supply (Continuous): DC voltage source for system input
  • DC Motor Block (Parameters):
    • Ra: 2.581 Ω
    • Rf: 281.3 Ω
    • La: 0.028 H
    • Lf: 156 H
  • PID Controller: Conventional speed control implementation
  • Measurement Block (100): Speed feedback system
  • Scope Display: Response monitoring

Working Principle:

The current system implements conventional PID control where speed feedback is compared with reference speed. The controller generates voltage signals based on the error, and the motor adjusts speed accordingly. Current metrics show settling time of 0.25s and torque ripple of 0.35%.

Future ANN Implementation:

The next phase will replace the PID controller with a neural network that learns optimal control patterns and adapts to load changes automatically. Expected improvements include faster response time and better accuracy.
