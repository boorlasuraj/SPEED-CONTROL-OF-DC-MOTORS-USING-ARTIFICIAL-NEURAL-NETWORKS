Speed Control of DC Motors using Artificial Neural Networks (ANN)

Project Overview

This project focuses on developing an advanced speed control system for DC motors using Artificial Neural Networks. The system aims to enhance precision, adaptability, and efficiency compared to conventional control methods. The ANN-based approach reduces torque ripple and eliminates the need for physical sensors while ensuring robustness against parameter variations.

Objectives

  • Develop an ANN-based control system for DC motor speed regulation
  • Implement real-time speed estimation without physical sensors
  • Reduce torque ripple and improve system efficiency
  • Achieve better dynamic response compared to conventional PI controllers
  • Create a robust system capable of handling parameter variations

MATLAB Simulation

Our implementation uses MATLAB/Simulink with the following components:

  • Neural Network Toolbox for ANN implementation
  • Dual-level neural architecture (ANN-train and ANN-control)
  • 5 HP, 240V, 1750 RPM DC motor model
  • Hysteresis current controller

Advantages

  • Improved speed control accuracy
  • Reduced settling time (0.11s vs 0.25s in PI control)
  • Lower torque ripple (0.124% vs 0.35% in PI control)
  • Sensorless operation capability
  • Better adaptation to load variations

Conclusion

The ANN-based speed control system demonstrates superior performance compared to conventional methods, achieving higher control quality (0.98), faster response times, and reduced torque ripple. The system proves particularly effective in applications requiring precise speed control under varying conditions.
